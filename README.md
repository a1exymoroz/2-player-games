# 2 Player Games

A collection of classic 2-player games built with Angular 17.

## Games Available

- **Tic Tac Toe** - Classic X and O game. Get three in a row to win!
- More games coming soon (Connect Four, Battleship)

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

### Build

Create a production build:

```bash
npm run build
```

## Features

- Modern, responsive UI
- Score tracking
- Smooth animations
- Mobile-friendly design
